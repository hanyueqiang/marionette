使用NPM和Webpack快速启动

Webpack是一个构建工具，它很容易地将依赖关系拉到一个单独的包中，以传递到 brower's <Script >tag。适用于Marionette和jQuery。

这里，我们用Webpack构建一个简单的Marionette实例。
